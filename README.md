# Firefly

A 75mm tinywhoop drone with a custom frame inspierd by [bubito](https://rotorriot.com/products/bubito-built-tuned) drone

---

## Features

- **Small** – fly indoors and outdoors with the 75mm wheelbase
- **0702 motors** 29kv!
- **AIO board** – (All In One) FC+ESC+OSD+VTX+RX
- **31mm props** – Best size i could fit in my frame.
